# TASK-1 (Калькулятор)

## Цель

Ознакомление с базовым функционалом языка программирования Go.
Написание простейшей программы.

## Задание

Необходимо создать консольное приложение - калькулятор,
которое будет выполнять простейшие математические операции

## Описание

1. Программа принимает на вход первый и второй операнд (целое число),
   а также символ операции, каждое из значений передается на отдельной
   строке. Символы, определяющие операции, представлены в таблице
   ниже:

| Операция | Описание            | Примеры |
| -------- | ------------------- | ------- |
| `+`      | Сложение операндов  | `a + b` |
| `-`      | Вычитание операндов | `a - b` |
| `*`      | Умножение операндов | `a * b` |
| `/`      | Деление операндов   | `a / b` |

2. Программа должна провести выбранную операцию над введенными
   числами, вывести результат на экран на новой строке и завершить
   выполнение.

3. Программа должна корректно обрабатывать ситуации, когда
   пользователь вводит некорректные данные:

- Если некорректно введен первый операнд - требуется вывести
  сообщение `Invalid first operand`;
- Если некорректно введен второй операнд - требуется вывести
  сообщение `Invalid second operand`;
- Если некорректно введена операция - требуется вывести
  сообщение `Invalid operation`;
- При делении на ноль программа должна выводить сообщение
  `Division by zero`.

## Примеры входных и выходных данных

```bash
# Входные данные
10
5
*
# Выходные данные
50
```

```bash
# Входные данные
15
5
%
# Выходные данные
Invalid operation
```

```bash
# Входные данные
not_number
5
-
# Выходные данные
Invalid first operand
```

```bash
# Входные данные
5
0
/
# Выходные данные
Division by zero
```
